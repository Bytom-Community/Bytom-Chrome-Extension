# chrome-extension-wallet

> Bytom Chrome Extension Wallet

## Build Setup

``` bash
# install dependencies
npm install

# serve with auto reload && import the **dist** folder in the Chrome browser
npm run dev

# build for production with minification && crx package
npm run build
```
